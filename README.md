About
IACLIN is a SaaS platform that uses AI agents to automate patient communication for clinics.

Unlike generic chatbots, IACLIN was built for healthcare workflows with a focus on operational efficiency, patient engagement, and a white-label experience for clinics.

Highlights
Automates appointment scheduling through WhatsApp using natural, clinic-friendly language.

Sends confirmations and reminders to help reduce no-shows.

Performs initial triage and routes urgent cases to the human team.

Supports patient reactivation and follow-up workflows.

Tech Stack
Monorepo: Bun workspaces

Frontend: TypeScript and Tailwind CSS

API Gateway: Elysia and TypeScript

AI Workflows: OpenAI, LangGraph, and Google Generative AI

Data Layer: PostgreSQL, Drizzle ORM, and Redis

Background Jobs: BullMQ

Integrations: gRPC, Stripe, and OpenAPI
